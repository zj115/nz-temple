# Documentation

This folder contains internal documentation for development, deployment, and system architecture.

## Structure

- `setup-guides/` - Authentication, CORS, and local development setup
- `deployment/` - Vercel and cloud deployment guides
- `aws/` - AWS S3 configuration, CORS setup, and scripts
- `database/` - Database schema and SQL setup files
- Root files - Calendar system documentation and quickstart guides

- `CALENDAR_SYSTEM_SUMMARY.md` - Overview of the Chinese almanac calculation system
